Data Science with Python

Working with large datasets using Python, Pandas, Matplotlib, and other data science tools.

Lesson 6: Machine Learning With Data

After these lessons you will be able to:

  • Explain what machine learning is and why it is part of the repertoire of data science methods
  • Configure machine learning tools and environments
  • Apply standard machine learning methods to create a model which performs image recognition of flower types using the flower image database from ImageNet.

For This Lesson You Will Need...

  • Python 2.7 or later
  • The following packages:
    • pandas
    • scikit-learn
    • tensorflow
    • keras
    • caffe
    • pytorch

These lessons are supported by the content at the following Github repository:

https://github.com/jsale/data_science_with_python
